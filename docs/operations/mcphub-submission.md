# MCP Hub (mcphub.ai) — Submission

[MCP Hub](https://www.mcphub.ai/) is a “Model Context Protocol Gateway” with a curated server directory, docs, and gateway service. It’s a different product from **mcphub.com**, **mcphub.dev**, and **mcphubs.com** (each has its own submit flow).

## Does mcphub.ai require action?

**Likely yes** if you want Syenite listed there. Their site doesn’t clearly document a public “Submit” URL, and they describe a **curated** collection, so listing may be by invite, form, or community request.

## What to do

1. **Check the site**
   - **[mcphub.ai/servers](https://www.mcphub.ai/servers)** — Look for “Add server”, “Submit”, or “Suggest a server”.
   - **Docs / MCP Development** — See if they explain how to add or suggest servers.
   - **Join Discussion** — Use their community/discord link and ask how to submit or suggest Syenite for the directory.

2. **If you find a form**
   - Use the same details as elsewhere: **Name:** Syenite, **URL:** https://syenite.ai or https://syenite.ai/mcp, **Description:** from README (DeFi interface for AI agents — swap, bridge, yield, lending via MCP).

3. **If there’s no form**
   - Request listing via their **Discussion** or contact (if listed in footer/About). One line: “We’d like to add our MCP server Syenite (https://syenite.ai) to the directory; how do we submit?”
