# PulseMCP Submission

[PulseMCP](https://pulsemcp.com) ingests from the **Official MCP Registry** daily and processes weekly. Syenite is already published there as **io.github.syenite-ai/syenite**, so it will appear on PulseMCP automatically within about a week.

## Optional: Direct submission (faster or to request adjustments)

If you want to be listed sooner or need changes to how Syenite appears on PulseMCP:

1. **Submit via form**
   - Go to **[pulsemcp.com/submit](https://pulsemcp.com/submit)**.
   - Choose **MCP Server**.
   - In the URL field, enter one of:
     - **https://syenite.ai** (standalone site)
     - **https://github.com/syenite-ai/syenite** (repo; they accept subfolders too).

2. **Email for adjustments**
   - For listing tweaks or if it’s been a week since Official Registry publish and you don’t see Syenite: **[hello@pulsemcp.com](mailto:hello@pulsemcp.com)**.
